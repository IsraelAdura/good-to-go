# Good to Go?

*Good to go* is a CLI app written in golang for getting up to date weather information.

## Installation

download and install [golang](https://golang.org)

## Usage
clone this directory

```bash
> go install
> cd $GOPATH bin
> good-to-go
```
or 

```bash
> go run main.go weather.go location.go
```
